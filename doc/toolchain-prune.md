# prune

Deletes unreferenced, or *orhpaned*, packages.

## Usage

	toolchain prune

## Example

```
PS C:\example> toolchain prune
Deleted: sha256:db2a58b317e90e537aa1e9b9ab4f1875689bcd9d25a20abdfbf96d3cb0a5ec45
Total reclaimed space: 141.24 MB
```
